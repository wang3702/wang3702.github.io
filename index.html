About Me

Xiao Wang is currently a research scientist at TikTok, focusing on multimodal learning. Previously, he was a postdoc at University of Washington, working on AI for biology. Prior to that, he obtained a CS Ph.D. degree from Purdue University, focusing on representation learning and AI for biology. He graduated with a bachelor's degree in CS from Xi'an Jiaotong University, Xi'an, China.

He used interned in Futurewei AI lab, Jingdong AI research, Meta AI (FAIR Labs). His research interests include multi-modal learning,AI for biology, representation learning and generative modeling.

    Open Source Projects

  • SLOT: SLOT is an Sample-specific Language Model Optimization algorithm at Test-time  
  • HiCFoundation:HiCFoundation is a generalizable Hi-C foundation model for chromatin architecture, single-cell and multi-omics analysis across species.  
  • CryoNeRF: CryoNeRF is a computational tool for homogeneous and heterogeneous (conformational and compositional) cryo-EM reconstruction in Euclidean 3D space.  
  • FlowTS: FlowTS is a rectified flow-based time series generation algorithm.  
  • DiffModeler:DiffModeler is a computational tool using diffusion model to automatically build full protein complex structure by taking native/AF2 single-chain structure as input.  
  • DeepMainMast: DeepMainMast is a computational tool using deep learning to automatically build full protein complex structure from cryo-EM map.  
  • AdPE: AdPE is a MIM based self-supervised learning method with adversarial embeddings.  
  • CryoREAD: Cryo_READ is a computational tool using deep learning to automatically build full DNA/RNA atomic structure from cryo-EM map.  
  • DAQ_Refine: DAQ_Refine is a protein structure refinement tool by DAQ-score and ColabFold.  
  • Asym Siam: Asym-Siam experimentally verified the importance of asymmetry for Siamese Representation Learning with obvious improvement.  
  • CaCo: CaCo is a state-of-the-art cooperative-adversarial contrastive learning method where both positive and negative samples are directly learnable.  
  • DAQ: DAQ is a software accesses the quality of protein models built from cryo-Electron Microscopy (EM) maps.  
  • OC_Finder: OC_Finder is a computational tool using deep learning for fully automated osteoclast segmentation, classification, and counting.
  • CoSeg: CoSeg is a self-supervised learning-based event boundary detection method.
  • CLSA: CLSA is a general contrastive learning framework by introducing the information from stronger augmentation.
  • AdCo: AdCo is an algorithm for effective self-supervised learning through adversarial training of negative examples.
  • GNN DOVE: GNN_DOVE is a software can evaluate the quality of protein-docking models using graph neural networks by reformulating protein structures as graphs.
  • Attention_AD: Attention_AD is a software that can distinguish active and inactive peptides for gene expression using Long Short Term Memory (LSTM).
  • Emap2sec+: Emap2sec+ is a software detects local structure information of proteins and DNA/RNA in cryo-EM maps using deep learning.
  • EnAET: EnAET is a software that benefits semi-supervised learning via self-trained ensemble auto-encoding transformations.
  • DOVE: DOVE is a software can evaluate the quality of protein-docking models using 3D neural networks.
